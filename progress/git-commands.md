# Git commans I've learned

## Basic commands
 - 'git init' - Initialize a new repository
 - 'git status' - check which files are staged/modified
 - 'git add .' - stage the whole directory 
 - 'git commit -m "xxxxx"' - commit the new additions to the repo with a message
 - 'git push' - push the changes to Github
 - 'git diff' - display changes made compared to previous version
