# VS Code Shortcuts I Use

## Navigation
- Cmd+P = Quick open any file
- Cmd+Shift+E = Focus Explorer panel
- Ctrl+Tab = Cycle through tabs

## Editing
- Cmd+D = Select next occurrence
- Option+↑/↓ = Move line up/down
- Shift+Option+↓ = Duplicate line
- Cmd+/ = Toggle comment
- Option+Click = Multiple cursors

## Search
- Cmd+F = Find in current file
- Cmd+Shift+F = Search across all files

## Git/Source Control
- Cmd+Shift+G = Open Source Control panel

## Other
- Cmd+S = Save
- Cmd+, = Settings
- Cmd+Shift+X = Extensions
